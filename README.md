# Kurdish-process-tracing-project
Qualitative research project using publicly available sources on the transformation of the Kurdish movement from armed resistance to democratic participation in Turkey
This repository contains the qualitative sources used for my process tracing research paper.


Files included:**
- `sources.docx` — a list of all primary and secondary sources used.
- `methods_and_access.docx` — a short explanation of how I accessed, organized, and analyzed the sources.

This project does not include any coding or statistical analysis.
